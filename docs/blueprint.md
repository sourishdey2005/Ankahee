# **App Name**: Ankahee

## Core Features:

- Anonymous Authentication: Implement anonymous user authentication using Supabase, generating a random username and device-based ID.
- Post Confession: Allow users to post text-based confessions with an optional mood tag and automatic deletion after 24 hours.
- Comment Anonymously: Enable users to comment anonymously on confessions, including nested comments.
- Automatic Deletion: Implement an auto-deletion system for posts and comments after 24 hours using Supabase Edge Functions.
- Realtime Feed: Display confessions in a realtime feed, sorted by newest first and excluding expired posts.
- Mood Tool: Leverage a generative AI tool to categorize and tag user posts based on sentiment analysis, suggesting the most relevant mood tag.

## Style Guidelines:

- Background: Dark background (#121212) to emphasize content and create a modern feel.
- Primary: Saffron (#FF9933) to reflect Indian heritage and create a sense of warmth.
- Accent: Deep purple (#4B0082) to add a touch of mystery and sophistication, providing a high-contrast highlight.
- Body font: 'Inter', a grotesque-style sans-serif for modern neutral legibility in long passages of text.
- Headline font: 'Space Grotesk' sans-serif for a modern, techy feel.
- Minimalist icons to represent mood tags and interactive elements, ensuring clarity and visual appeal.
- Card-based layout for confessions with a floating action button for creating new posts, optimizing user engagement.
- Smooth micro-interactions and subtle animations throughout the app to enhance user experience.
